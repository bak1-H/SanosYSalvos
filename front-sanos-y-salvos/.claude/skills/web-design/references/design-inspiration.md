# Recursos e Inspiración de Diseño Web

## Galerías y referencias visuales
- [Dribbble](https://dribbble.com) — diseño UI/UX de alta calidad
- [Behance](https://behance.net) — portfolios y proyectos completos
- [Awwwards](https://awwwards.com) — sitios premiados por diseño web
- [Godly](https://godly.website) — curación de diseño web moderno
- [Mobbin](https://mobbin.com) — patrones de UI mobile y web reales

## Paletas de color
- [Coolors](https://coolors.co) — generador y explorador de paletas
- [Realtime Colors](https://realtimecolors.com) — preview de paletas en UI real
- [Huemint](https://huemint.com) — paletas generadas con IA
- [Tailwind Colors](https://tailwindcss.com/docs/customizing-colors) — escala de colores lista para usar

## Tipografía
- [Google Fonts](https://fonts.google.com) — fuentes gratuitas
- [Fontpair](https://fontpair.co) — combinaciones de fuentes
- [Typescale](https://typescale.com) — escalas tipográficas visuales

## Iconos
- [Lucide](https://lucide.dev) — open source, estilo limpio
- [Heroicons](https://heroicons.com) — por el equipo de Tailwind
- [Phosphor Icons](https://phosphoricons.com) — muy completo, múltiples estilos
- [Iconify](https://iconify.design) — acceso a múltiples sets

## Ilustraciones y assets
- [Undraw](https://undraw.co) — ilustraciones SVG personalizables
- [Storyset](https://storyset.com) — ilustraciones animadas
- [Unsplash](https://unsplash.com) — fotos gratuitas de alta calidad
- [Pexels](https://pexels.com) — fotos y videos gratuitos

## Herramientas de diseño
- [Figma](https://figma.com) — estándar de la industria para UI/UX
- [Excalidraw](https://excalidraw.com) — wireframes rápidos a mano alzada
- [Whimsical](https://whimsical.com) — flowcharts y wireframes

## Gradientes y efectos
- [CSS Gradient](https://cssgradient.io) — generador de gradientes CSS
- [Glassmorphism](https://hype4.academy/tools/glassmorphism-generator) — efecto glass
- [Neumorphism](https://neumorphism.io) — efecto neumorfismo

## Design systems de referencia
- [Tailwind UI](https://tailwindui.com) — componentes de Tailwind
- [shadcn/ui](https://ui.shadcn.com) — componentes React accesibles
- [Radix UI](https://radix-ui.com) — primitivas accesibles
- [Material Design 3](https://m3.material.io) — design system de Google
- [Apple HIG](https://developer.apple.com/design/human-interface-guidelines/) — guías de Apple
